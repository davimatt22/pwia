Manual das Tags HTML

A tag "body" no HTML é usada para envolver o conteúdo principal de uma página web. Ela é onde você coloca os elementos visíveis para o usuário, como textos, imagens, links, tabelas, e outros elementos interativos.

Tag "H1"

H1: É uma tag usada para titulos principais, ou a parte do cabeçalhos da pagína Ela tem um grande significado tanto para os visitantes quanto para os motores de busca (SEO).

Tag "Head"

A tag "head" engloba todas as informações de configuração e propriedades do documento HTML. Esses dados não são visíveis para o usuário diretamente na interface da página, mas desempenham um papel fundamental para o navegador e os motores de busca. Dentro do "head", você pode incluir elementos como o título da página, links para estilos CSS, scripts JavaScript, e informações que ajudam a otimizar a página para SEO (Search Engine Optimization).

Tag "tiltle"

O "tiltle" é a parte que se nomeia o site, é o Titulo da pagina

Tag "p"

"p" ultilizamos como paragrafo da pagina

Tag "DIV"

"DIV", A tag

é usada para criar blocos de conteúdo dentro de uma página, permitindo que você agrupe diferentes partes do conteúdo. Ela não altera o comportamento visual dos elementos internos, mas oferece uma maneira de aplicar estilos, como margens, padding, cores de fundo, entre outros, a um conjunto de elementos.

Tag "BUTTON"

"button" A tag no HTML é usada para criar botões clicáveis em uma página web. Esses botões podem ser utilizados para executar ações quando o usuário interage com eles, como enviar um formulário, executar um script, navegar entre páginas, entre outras funcionalidades.

Tag "IMG"

"img" É ultilizada para iserir imagens no site criado. E também tem o alt, O atributo alt fornece uma descrição da imagem. Isso é importante tanto para a acessibilidade (ajuda usuários com deficiência visual a entender o conteúdo da imagem)
Contem também Width e Height, que significa a largura e o comprimento

Tag "INPUT"

"input": A tag cria campos de entrada nos formulários, onde o usuário pode inserir dados. Dependendo do tipo de entrada desejado, você define o tipo do input usando o atributo type
