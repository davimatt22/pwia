Exercícios de Lista (JavaScript - Verifique o Console)

Os resultados de cada exercício foram enviados para o console do navegador.

Para vê-los:

  1. Clique com o botão direito em qualquer lugar da página.
  2. Selecione "Inspecionar" ou "Inspecionar Elemento".
  3. Vá para a aba "Console" nas Ferramentas do Desenvolvedor que se abrirão.
